# Sound Files Setup

To enable sound effects, please place the following MP3 files in this directory (`public/sounds/`):

- `intro.mp3`         (Played during INTRO phase)
- `read_question.mp3` (Played when question appears)
- `countdown.mp3`     (Played during 3-2-1 countdown)
- `start.mp3`         (The "Go!" sound)
- `thinking.mp3`      (BGM while answering)
- `timeup.mp3`        (End of time sound)
- `correct.mp3`       (Played when answer is revealed)
- `ranking.mp3`       (Played during ranking display)

*Note: You can use any audio format supported by browsers, but MP3/WAV is recommended.*
